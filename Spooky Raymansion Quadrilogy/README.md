# The Spooky Raymansion Quadrilogy Manual Archipelago

A manual apworld for the [_elusive_](https://gamejolt.com/games/thespookyraymansion/447686) [Spooky](https://gamejolt.com/games/raymansion2/548803) [Raymansion](https://gamejolt.com/games/raymansion3/658338) [games](https://gamejolt.com/games/returntothespookyraymansion/848202)!
This was made as a joke, the apworld is best played as a side game rather than a main game.

Your main goal is to save Globox from Noseless Rayman!
- Get the good ending on Return to the Spooky Raymansion

The apworld expects the player to have a completed save for Raymansion 2, Raymansion 3 and Return to Raymansion.
These will be provided in it's respective folder.

World progression is locked behind Lums. They're required to access different levels progressively in the quadrilogy.

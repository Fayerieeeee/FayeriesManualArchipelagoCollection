A manual apworld for Pac-Man Arrangement! You can use any version of the game to play!
[Pac-Man Museum+](https://store.steampowered.com/app/1665130/PACMAN_MUSEUM/) is the current modern day release of Pac-Man Arrangement so it's best to get that for legal reasons.

Your main goal is to defeat the Ghost Gang! The many worlds of Pac-Man Arrangement are locked because of the Ghost!
- Defeat the Ghost Gang

The apworld expects the player to start at World 0 - Toy Box World
World progression is locked by progressive world unlocks, getting one grants the player to access the next world accordingly. 

# üìù Progression Items
- Progressive World (6): Last Progressive World item is required to access the final boss.
- Progressive Fruit Bonus (6): Required to send freshly eaten fruit bonuses as checks.
- Progressive S-Dot (5): Required to fight off the Ghost Gang's robot.
- Credits (10): Each credits received allows the player to use them per run, currently can't be toggled off but can be safely ignored if you prefer unlimited credits.
